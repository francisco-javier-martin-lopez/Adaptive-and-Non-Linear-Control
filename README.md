# Adaptive-and-Non-Linear-Control

This course consisted in two projects:

First one consisted in the desing of a baseline attitude and position controler of a hexarotor UAV with known characteristic. Afterwards, by addition of a package with uncertain mass and possition, to design an adaptive controler that ensures the correct trajectory tracking.

Second one consisted in a small presentation with an overview in the state of the art of some of the concepts about autonomous control given in the course. In this case, it was done about C* path planning algorithms.
